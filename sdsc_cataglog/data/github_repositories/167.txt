repo_name: PyCEbox
link: https://github.com/AustinRochford/PyCEbox
description: PyCEbox is a Python implementation of individual conditional expecation plots inspired by.R's ICEbox, which is available on PyPI and can be installed with pip install.pycebox. For easy development and prototyping using IPython notebooks, a Docker.environment is included. To run an IPython notebook with access to your.development version of pycebox, run PORT=8889 sh ./start_container.sh. A.Jupyter notebook server with access to your development version of `pycebox`.should be available at `http://localhost:8889/tree`. For details of `pycebox`'s API, consult the documentation.
