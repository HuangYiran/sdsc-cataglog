repo_name: mybinder.org-deploy
link: https://github.com/jupyterhub/mybinder.org-deploy
description: This repository is the deployment, configuration, and Site Reliability documentation files for the public mybinder.org service. Please note that this repository is participating in a study into sustainability of open source projects, with data being gathered about this repository for approximately the next 12 months. If you wish to deploy your own Binder instance, please do not use these files. Instead, review the BinderHub documentation and the `jupyterhub/binderhub` repo to set up your deployment. The Site Reliability Guide contains the collected wisdom of operating mybinder.org, as well as other useful information.
