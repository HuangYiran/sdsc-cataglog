description: Hydrosphere Serving is a cluster for deploying and versioning your machine learning models in production. You can refer to our documentation to see tutorials, check out example projects, and learn about all features of Hydrosphere. There are two main ways of installing Hydrosphere. Before installing Hydrosphere Serving, please install its prerequisites. To install Hydrosphere Serving, follow the instructions below. To reach the cluster, port-forward `ui` service locally. Kubectl port-forward -n hydrosphere svc/serving-ui 8080:9090. To check installation, open http://localhost:8080/. **Note**, other installation options are described in the documentation.
