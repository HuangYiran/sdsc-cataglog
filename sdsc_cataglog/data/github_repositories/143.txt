link: https://github.com/argoproj/argo-workflows
description: Argo Workflows is an open source container-native workflow engine for orchestrating parallel jobs on Kubernetes. Argo Workflows is implemented as a Kubernetes CRD (Custom Resource Definition). It has many features such as the ability to automate any workflow, host and manage packages, find and fix vulnerabilities, create instant dev environments, write better code with AI, and collaborate outside of code. Argo Workflows is used by about 200+ organizations and many projects. The project is governed by the CNCF Code of Conduct and has monthly community meetings and blogs.
