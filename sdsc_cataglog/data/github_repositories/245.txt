description: ModelDB is an open-source system to version machine learning models including their ingredients code, data, config, and environment, and to track ML metadata across the model lifecycle. Use ModelDB in order to automate any workflow, host and manage packages, find and fix vulnerabilities, create instant dev environments, write better code with AI, and collaborate outside of code. If you are looking for a hosted version of ModelDB, please reach out at modeldb@verta.ai. This version of ModelDB is built upon its predecessor from CSAIL, MIT and the previous version can be found on Github. The ModelDB project is now maintained by Verta.ai. The official documentation for ModelDB can be found here, and for Getting Started guides, Tutorials, and API reference check out our docs. To report a bug, file a documentation issue, or submit a feature request, please open a GitHub issue. For help, questions, contribution discussions, and release announcements, please join us on Slack.
