description: The dataset called "mfeat-factors" comprises a collection of handwritten digit images from 0 to 9. Each image is represented by 76 features extracted from the Fourier transform of the original image. These features include the intensity values at different frequencies and orientations. The dataset contains a total of 2,000 samples, with each sample labeled based on the digit it represents. The data is provided in the ARFF format, which is commonly used for machine learning tasks. This dataset can be utilized for various tasks such as digit recognition, image classification, and feature extraction.
