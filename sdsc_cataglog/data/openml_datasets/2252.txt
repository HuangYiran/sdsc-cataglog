name: threeOf9
dataset_id: 40690.0
description: The dataset threeOf9 consists of optical barcodes representing the characters 0-9. These barcodes were generated using the Code 39 symbology, a widely used barcode standard. The dataset consists of 10,000 images, each containing one barcode character. The images are in grayscale and have a resolution of 32x32 pixels. The dataset is split into a training set of 7,000 images and a test set of 3,000 images. The dataset aims to provide a comprehensive collection of barcode characters that can be useful for tasks such as machine learning, computer vision, and optical character recognition research and development.
