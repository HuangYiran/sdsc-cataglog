description: The dataset "kc1-top5" is a collection of software metrics for five open-source software projects. These metrics represent various aspects of the software development process and help evaluate the quality and maintainability of the projects. The dataset contains a total of 4179 instances, with each instance representing a software module. The attributes in the dataset include metrics like lines of code, cyclomatic complexity, and number of variables. Each instance is labeled as either defective or non-defective, based on whether the software module contains any known defects. The dataset aims to aid in software quality prediction and defect detection research.
