name: USPS
dataset_id: 41964.0
description: The dataset USPS contains handwritten digits and their corresponding labels (0-9) for training and testing purposes. It consists of a total of 9298 images, with each image being 16x16 pixels in size. The training set encompasses 7291 samples, while the test set comprises 2007 samples. The dataset offers a variety of digits, with each digit appearing different number of times. It is a popular dataset for testing and evaluating various machine learning algorithms, particularly for handwriting recognition tasks.
